# AI Model Manager

A web application to manage AI models where users can **add, view, purchase, and track AI models**. Built with the MERN stack and MongoDB Atlas for backend storage.

---

## Features

- Users can **add new AI models** with details like framework, use case, dataset, and image.  
- Users can **view model details**.  
- Users can **purchase models**, which increments a purchase count.  
- Users can **see their own purchases** in a personal dashboard.  
- creator can **edit or delete models** they added.  
- Fetch **latest models** sorted by creation date.  

---

## Tech Stack

| Layer      | Technology                        |
|------------|----------------------------------|
| Frontend   | React, React Router, TailwindCSS  |
| Backend    | Node.js, Express.js               |
| Database   | MongoDB Atlas                     |
| Auth       | Firebase / Custom Context API     |
| Deployment | Vercel / Render / Localhost       |
